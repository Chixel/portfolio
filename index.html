PROJECTS

Anime Notifier

Anime Notifier is an extension for chromium-based browsers such as chrome and opera, as well as firefox. It's purpose is to notify you when an anime you are following has been released on the website animekisa.tv, it does so by webscraping said website every 5 minutes and whenever you start up your browser, getting the latest information from it.

Roomba discord bot

Roomba is a discord bot made in node js using the discord.js api. It's purpose is to allow server owners to sync up different text channels on completely different discord servers. It also allows users to communicate anonymously with a setting accessed through the "/anonymous true" command.
